# 📱 SchemeConnect

SchemeConnect is a native iOS application developed using **Swift** and **Xcode** that helps users discover government welfare schemes in one place.

---

## ✨ Features

- 📚 Browse government schemes
- 🎓 Student schemes
- 👨‍🌾 Farmer schemes
- 👵 Pension schemes
- 💼 Employment schemes
- 🔍 Easy navigation
- 🔗 Official government portal links

---

## 🛠️ Tech Stack

- Swift
- Xcode
- iOS
- UIKit / SwiftUI

---

## 🚀 Installation

1. Clone the repository

```bash
git clone <your-github-link>
```

2. Open in Xcode

3. Run the project
